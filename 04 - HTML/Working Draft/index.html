The Global Digital Divide

An analysis of internet usage across the world

Project Overview

For our project, we want to analyse global disparities in regards to access to resources such as the Internet and the opportunities derived from such access. We have relied on a number of resources as listed below:

  • World bank API https://datahelpdesk.worldbank.org/knowledgebase/articles/898581-api-basic-call-structures
  • Web index https://thewebindex.org/downloads/
  • Our World in data https://ourworldindata.org/internet#all-charts-preview
  • Compare the market https://www.comparethemarket.com/broadband/content/global-broadband-index/
  • Internet World Stats https://www.internetworldstats.com/stats1.html
